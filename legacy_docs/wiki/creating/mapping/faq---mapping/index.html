FAQ - Mapping

  • Why can't I copy&paste or use heightmaps?
    Some commands can't be used in multiplayer edit mode - these include copy&paste and heightmap editing. For this, you need to be the only person editing the map.

    To do so, select 'Request private edit mode' in the menu, when connected to a server (and when no one else is currently there). Once in private edit mode, you can use copy&paste, heightmaps, and so forth.
