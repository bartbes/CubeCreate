Mapping

Mapping is the art of creating a 3D 'world'. It involves shaping that world, texturing it, and placing entities (props, etc.). Mapping is one of the two main disciplines in creation, the other being Scripting.

Basics

For an introduction to mapping, see the mapping tutorial. Another good place to start is the Sauerbraten editing intro (but, be aware that editing in Syntensity is also different in some ways, as summarized here).

Advanced

See the advanced Sauerbraten documentation,
  • editref
  • models
Note however that map scripts and model scripts are done in JavaScript in Syntensity, not cubescript. However, aside from notational differences, things are mostly similar.
