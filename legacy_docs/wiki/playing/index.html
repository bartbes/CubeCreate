Playing

Terms

  • Activity: Usually a game, but can be any sort of event.
  • Server Instance: The 'game servers', that can run activities. The same activity can be run on several server instances at once, each as a separate game.
  • Client: The client program you run on your computer, that connects to Syntensity server instances

Overview

  • Syntensity always has a lot of server instances running. Some might be actively running activities, and some might be idle.
  • You can look at the list of active instances on the Syntensity website, and can pick an instance to join. You can then run the client and enter that instance.
  • If no instance is running the activity you want, or you just want to run a particular activity in a new instance, you can requisition a server instance. Requisitioning means asking Syntensity to find an available server instance, and to tell it to run that activity
