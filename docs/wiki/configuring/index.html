Configuring

This page gives some tips on how to configure the client program.

Cubescript

Using cubescript you can change keybindings and a lot of other things. Generally you can make these changes by adding code to autoexec.cfg, which is in your home directory (the home directory is intensityengine_client or .intensityengine_client, in your operating system user's home directory - under Documents and Settings on XP, Users on Vista, ~ on Linux, etc.). autoexec.cfg is run every time you start the client. In the attachments below are some some example scripts you can add:
  • Play in thirdperson, edit in first person (can easily change it to do the reverse, etc.)
  • Hold right CTRL to move fast in edit mode (nice for big maps)
