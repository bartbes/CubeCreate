.. _glossary:

Glossary
========

.. glossary::

    Activity
        A legacy term for what is called a map now.

    Client
        The client program you run on your computer, that connects to
        CubeCreate server instances.

    Map
        Usually a game, but it can be any sort of event.

    Server Instance
        The 'game servers', that can run maps. The same map can be run on
        several server instances at once, each as a separate game.
